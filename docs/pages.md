# 页面原型说明

## 1. 今日页

这是第一版主页面，回答一个问题：**今天该怎么过。**

### 主要模块

1. 今日概况
   - 今天任务数
   - 已完成数
   - 不做清单守住数
   - 是否完成一次真实交流

2. 今天要做
   - 今日任务卡片
   - 任务完成按钮
   - 上下调整顺序
   - 拆最小下一步输入框
   - 开始 25 分钟专注按钮

3. 今天不做
   - 今日边界清单
   - 点击即可标记“已守住”

4. 最小下一步
   - 当前选中的今日任务
   - 当前最小动作

5. 状态记录
   - 状态模板选择
   - 诱因输入
   - 应对方式输入
   - 结果选择

6. 与人交流
   - 是否已经进行一次真实交流
   - 备注联系对象或计划

7. 放松窗口
   - 是否已经解锁
   - 剩余时段
   - 推荐的低刺激放松内容

### 核心交互

- 点击任务后，右侧显示当前最小下一步
- 每个步骤都可以直接启动番茄钟
- 今日页尽量避免跳转，强调一屏内推进

## 2. 任务池页

### 主要模块

1. 新增任务表单
   - 任务名
   - 主动任务 / 固定生活任务
   - 固定任务提醒时间

2. 任务池列表
   - 主动任务
   - 固定生活任务
   - 一键放进今天

3. 长期规则预览
   - 要做 / 不做规则标签

### 核心交互

- 新建一个任务后，立即可放入今天
- 固定生活任务与主动任务统一管理，但标签区分明显

## 3. 模板页

### 主要模块

1. 每日模板
   - 今日核心任务数
   - 今日生活任务数
   - 今日不做条数
   - 今日交流提示
   - 放松窗口分钟数

2. 每周模板
   - 本周 3 个方向
   - 本周最易失守场景
   - 本周交流目标
   - 本周休息安排

3. 提醒与干预
   - 干预等级
   - 黑名单目标
   - 是否开启鼓励提醒

### 核心交互

- 所有模板为固定槽位，不提供无限自定义
- 保存后立即影响当前使用逻辑

## 4. 复盘页

### 主要模块

1. 今日复盘表单
   - 今天完成了什么
   - 今天失守了什么
   - 今天最常进入什么状态
   - 明天第一步是什么

2. 今日困难列表
   - 困难类型
   - 说明
   - 下一步动作

3. 周模板预览
   - 本周方向
   - 失守场景
   - 交流目标

### 核心交互

- 晚间复盘尽量简短，不做长篇反思
- 重点在于帮助用户准备“明天第一步”

## 5. 全局悬浮番茄钟

### 主要模块

- 倒计时
- 当前任务
- 当前最小动作
- 结束并记录
- 取消本轮

### 核心交互

- 番茄钟全局悬浮，用户切换页面也能看到
- 时间结束后自动打开记录弹层

## 6. 番茄结束弹层

### 主要模块

- 本轮是否完成
- 是否顺手完成当前最小任务
- 困难类型
- 具体卡点
- 下一步准备怎么解决

### 核心交互

- 如果未完成，鼓励用户立刻写下下一步，而不是退出
- 提交后自动把下一步加回任务里
